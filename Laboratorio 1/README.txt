Instrucciones Generales:

Para que el codigo pueda ser utilizado correctamente se debe tener en cuenta lo siguiente:

- Debe arrastrar todos los archivos .c junto con el MAKEFILE a una carpeta cualquiera dentro de su sistema.

- Debe colocar los archivos .txt (Sopas de letras) de prueba en la carpeta donde coloco los archivos mencionados en el punto anterior (Asegurarse de que los archivos .txt no esten corruptos)

Ya hecho lo anterior, debe abrir la consola y dirigirse con el comando "cd" a la carpeta que contiene todos los archivos mencionados anteriormente. Una vez dentro de la carpeta en la consola debe escribir el comando "make" o "make all" el que compilara los archivos T1SO.c y T1optimo.c

Ahora bien, para probar los archivos debe escribir en consola lo siguiente:

Para probar el archivo T1SO.c:
make run_t1

Para probar el archivo T1optimo.c:
make run_t1op

Esto se hizo así para no tener que juntar ambos codigos en uno solo.

Escribiendo uno de estos comandos, el codigo no pedira nada al usuario, por lo que se pueden mirar los tiempos de ejecucion de pada palabra con su respectiva horientacion sin problema.
